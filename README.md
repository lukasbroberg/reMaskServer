# reMaskServer
This repository contains the server for reMask. The server is essential for reMask to run. 
# How to run
Prerequisites: have node.js installed (https://nodejs.org/en)
Open the project folder in your preferred IDE.
Open up the terminal.
Install the necessary dependencies using the node package manager, by typing "npm install".
Type "node index.js" in the terminal, to start the server.
